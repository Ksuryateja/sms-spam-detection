# sms-spam-detection
<br>
data is available at https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection
<br>
The steps employed were
<br>

  1. Text Processing
  
    - Getting rid of punctuations.
    - Getting rid of stop words.
    - Tokenization.
2. Bag of Words
3. Naive Baye's Classifier.
